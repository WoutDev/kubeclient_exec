# Kubeclient_exec
A simple gem adding `exec_pod` and `cp_pod` functionality to [Kubeclient](https://github.com/ManageIQ/kubeclient).

# Installation
Add this line to your Gemfile:
```ruby
gem 'kubeclient_exec'
```

And then install the gem:
> bundle
